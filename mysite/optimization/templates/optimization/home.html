This website provides tools to solve optimization problems using different mathematical programming techniques such as Simplex, Graphical, and MODI Methods.

Simplex Method

The Simplex Method is an algorithm used for solving linear programming problems. It iteratively improves a feasible solution by moving along the edges of the feasible region to reach the optimal solution.

Steps to Solve:

  1. Formulate the objective function and constraints in standard form.
  2. Set up the initial simplex tableau.
  3. Identify the pivot column (most negative value in the bottom row).
  4. Identify the pivot row (smallest ratio of right-hand side to pivot column value).
  5. Perform row operations to update the tableau.
  6. Repeat until no negative values remain in the bottom row.

Graphical Method

The Graphical Method is used for solving linear programming problems with two decision variables. It provides a visual representation of feasible solutions.

Steps to Solve:

  1. Plot all constraints on a graph as linear equations.
  2. Identify the feasible region where all constraints overlap.
  3. Find the corner points of the feasible region.
  4. Evaluate the objective function at each corner point.
  5. Select the point that maximizes or minimizes the objective function.

MODI Method

The MODI (Modified Distribution) Method is used for solving transportation problems. It optimizes the initial basic feasible solution found using methods like the Northwest Corner Rule or Least Cost Method.

Steps to Solve:

  1. Find an initial feasible solution using a method like Northwest Corner.
  2. Compute the dual variables (U, V) for each row and column.
  3. Calculate opportunity costs for non-basic variables.
  4. If all opportunity costs are non-negative, the solution is optimal.
  5. If negative values exist, adjust allocations along a closed loop and repeat.
